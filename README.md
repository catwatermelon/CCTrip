# CCTrip
基于高德地图的出行个人练习项目

技术栈: mysql+koa2.js+vue.js+echarts.js，基于高德地图服务

使用jwt+axios拦截器以及路由守卫实现token的验证和拦截

页面的出行和历史模块已完成

首页由于定位不准不好测量，暂时先放着

个人页面仅仅是完成了账号、用户名以及头像上传功能，表格采用的是静态数据
